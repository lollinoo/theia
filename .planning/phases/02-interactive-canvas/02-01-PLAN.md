  <action>
Create a new React project in the `frontend/` directory. Do NOT use `npm create vite` interactively -- write all files directly.

**package.json:** React 18, react-dom, typescript, @types/react, @types/react-dom, vite, @vitejs/plugin-react, tailwindcss, postcss, autoprefixer. Include scripts: dev (vite --host), build (tsc -b && vite build), preview (vite preview).

**vite.config.ts:** Configure with react plugin. Set server.host to '0.0.0.0' and server.port to 3000 for Docker. Add proxy: `/api` routes to `http://backend:8080` (Docker service name) with changeOrigin true. For local non-Docker dev, also support VITE_API_URL env override.

**tailwind.config.js:** Content paths to `./src/**/*.{ts,tsx}`. Set darkMode to 'class'. Extend theme with custom colors for the charcoal dark palette:
- bg-canvas: '#2d2d3d' (main canvas background)
- bg-surface: '#363647' (card/panel surfaces)
- bg-elevated: '#3f3f53' (hover/elevated states)
- border-subtle: '#4a4a5e' (borders)
- text-primary: '#e1e8ed' (primary text)
- text-secondary: '#8899a6' (secondary/muted text)
- accent: '#00d4ff' (accent/highlight color)
- status-up: '#00c853' (green)
- status-down: '#ff1744' (red)
- status-probing: '#ffc107' (yellow)
- status-unknown: '#657786' (gray)

**postcss.config.js:** Standard tailwindcss + autoprefixer plugins.

**index.html:** Minimal HTML with div#root, title "MikroTik Theia", dark class on html element.

**src/index.css:** Tailwind directives (@tailwind base, components, utilities). Set body background to bg-canvas color, text to text-primary. Use `@layer base` to set default dark background and text on body/html.

**src/App.tsx:** Simple shell component rendering a full-viewport div with bg-canvas background. Show centered text "MikroTik Theia" with the accent color gradient (matching placeholder style). This will be replaced with the canvas in Plan 03.

**src/main.tsx:** Standard React 18 createRoot render.

**src/vite-env.d.ts:** Vite client types reference.

**tsconfig files:** Standard Vite React TypeScript config. Target ES2020, strict mode enabled, jsx react-jsx.
