<objective>
Remove the per-device interfaces array from GET /api/v1/devices JSON payload and migrate all frontend consumers to lazy-fetch interfaces on demand.

Purpose: At scale (100+ devices x 25 interfaces each), the interfaces block adds ~475KB to every device list response. Removing it from the list endpoint and fetching on-demand reduces initial page load payload significantly.

Output: Slimmer device list API response, three frontend components migrated to lazy fetch, edge rendering handles empty interfaces gracefully.
</objective>
