Create domain types in internal/domain/device.go:
- `DeviceType` enum: Router, Switch, AP, Unknown
- `DeviceStatus` enum: Up, Down, Probing, Unknown
- `SNMPv2cCredentials` struct: Community string
- `SNMPv3Credentials` struct: Username, AuthProtocol, AuthPassword, PrivProtocol, PrivPassword, SecurityLevel
- `SNMPCredentials` struct: Version (2c or 3), V2c *SNMPv2cCredentials, V3 *SNMPv3Credentials
- `Interface` struct: ID (UUID), DeviceID, IfIndex int, IfName string, IfDescr string, Speed int64, AdminStatus string, OperStatus string, CreatedAt, UpdatedAt
- `Device` struct: ID (UUID), Hostname, IP, SNMPCredentials, DeviceType, Status, SysName, SysDescr, SysObjectID, HardwareModel, Managed bool (true=user-added, false=discovered placeholder), Tags map[string]string, Interfaces []Interface, CreatedAt, UpdatedAt
- `DeviceRepository` interface: Create, GetByID, GetAll, Update, Delete, GetByIP (for dedup on neighbor discovery)

Create internal/domain/link.go:
- `Link` struct: ID (UUID), SourceDeviceID, SourceIfName, TargetDeviceID, TargetIfName, DiscoveryProtocol (LLDP/CDP), CreatedAt, UpdatedAt
- `LinkRepository` interface: Create, GetByDeviceID, GetAll, Delete, Upsert (for rediscovery)

Create internal/domain/settings.go:
- `Settings` struct with key-value pairs for: PrometheusURL, GrafanaURL, PollingIntervalSeconds, SNMPWorkerPoolSize, SNMPTimeoutSeconds, SNMPRetries
- `SettingsRepository` interface: Get(key), Set(key, value), GetAll

Create internal/config/config.go:
- `Config` struct: ListenAddr string, DBPath string, LogLevel string
- `func Load(path string) (*Config, error)` — reads YAML file, then applies env var overrides: THEIA_LISTEN_ADDR, THEIA_DB_PATH, THEIA_LOG_LEVEL
- Default values: ListenAddr ":8080", DBPath "./data/theia.db", LogLevel "info"

Create config.example.yaml with documented fields.

Create cmd/theia/main.go:
- Loads config via config.Load (default path: "config.yaml", overridable via --config flag or THEIA_CONFIG env var)
- Opens SQLite database
- Placeholder log line "Theia server starting on {addr}"
- Does NOT start HTTP server yet (Plan 03 adds that)

Dependencies: go get github.com/google/uuid, github.com/mattn/go-sqlite3, gopkg.in/yaml.v3
