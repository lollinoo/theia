## Decisions Made
- Only interface names are editable via PUT; to change devices, delete and recreate the link
- Interface filter excludes names starting with "lo", "Null", or empty (no loopback/null/empty clutter)
- in_use annotation derived from GetByDeviceID result set (no dedicated DB column needed)
- GetBySysName added for O(1) LLDP neighbor resolution (cleaner than iterating GetAll)
- LLDP auto-link uses Upsert to ensure idempotent behavior on re-probe
