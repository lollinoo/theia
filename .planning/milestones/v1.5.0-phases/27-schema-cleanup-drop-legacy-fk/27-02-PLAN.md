The current eligibility check at line 103 uses `if (!d.ssh_profile_id)` to skip devices without an SSH profile. Since `ssh_profile_id` no longer exists on the Device type, this must be replaced with a check against credential profile assignment.

**Approach:** Use `fetchDeviceCredentialProfiles(d.id)` to check if the device has at least one assigned credential profile. However, calling this per-device inside the synchronous `handleStart` loop would be awkward. Instead:

1. Add an import for `fetchDeviceCredentialProfiles` from `'../../api/client'`.
2. Make `handleStart` async. Before building the `initial` entries array, pre-fetch credential profiles for all devices in parallel:
```typescript
const handleStart = async () => {
    setPhase('running');
    setError('');

    // Pre-fetch credential profile assignments for all devices in parallel
    const profileResults = await Promise.allSettled(
      devices.map((d) => fetchDeviceCredentialProfiles(d.id))
    );
    const deviceHasProfile = new Map<string, boolean>();
    for (let i = 0; i < devices.length; i++) {
      const result = profileResults[i];
      deviceHasProfile.set(
        devices[i].id,
        result.status === 'fulfilled' && result.value.length > 0
      );
    }

    // Build entries â€” pre-check eligibility
    const initial: DeviceEntry[] = devices.map((d) => {
      const name = getDeviceName(d);
      if (!d.backup_supported) {
        return { deviceId: d.id, deviceName: name, phase: 'skipped' as const, reason: 'backup not supported for this vendor' };
      }
      if (!deviceHasProfile.get(d.id)) {
        return { deviceId: d.id, deviceName: name, phase: 'skipped' as const, reason: 'no credential profile assigned' };
      }
      return { deviceId: d.id, deviceName: name, phase: 'checking' as const };
    });

    setEntries(initial);
    entriesRef.current = initial;
    // ... rest of function unchanged
```

Note the reason message changes from `'no SSH profile assigned'` to `'no credential profile assigned'` to match the new system.
