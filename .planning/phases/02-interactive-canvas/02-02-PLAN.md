**internal/domain/position.go:** Define DevicePosition struct and PositionRepository interface:
```go
type DevicePosition struct {
    DeviceID  uuid.UUID `json:"device_id"`
    X         float64   `json:"x"`
    Y         float64   `json:"y"`
    Pinned    bool      `json:"pinned"` // true = manually placed, false = auto-layout
    UpdatedAt time.Time `json:"updated_at"`
}

type PositionRepository interface {
    GetAll() ([]DevicePosition, error)
    SaveAll(positions []DevicePosition) error
    DeleteByDeviceID(deviceID uuid.UUID) error
}
```

**internal/repository/sqlite/migrations.go:** Append a new migration to the migrations slice:
```sql
CREATE TABLE IF NOT EXISTS device_positions (
    device_id TEXT PRIMARY KEY,
    x REAL NOT NULL DEFAULT 0,
    y REAL NOT NULL DEFAULT 0,
    pinned INTEGER NOT NULL DEFAULT 0,
    updated_at DATETIME NOT NULL,
    FOREIGN KEY (device_id) REFERENCES devices(id) ON DELETE CASCADE
)
```

**internal/repository/sqlite/position_repo.go:** Implement PositionRepository:
- GetAll: SELECT all rows, return []DevicePosition
- SaveAll: Use a transaction with INSERT OR REPLACE for each position
- DeleteByDeviceID: DELETE WHERE device_id = ?

Follow the same patterns as the existing device_repo.go and link_repo.go (constructor NewPositionRepo(db), same error handling style).
